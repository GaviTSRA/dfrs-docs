# Actions
Actions represent the following codeblocks:
- Player Action
- Entity Action
- Game Action
- Select
- Control
<br/>
They are used with the following syntax:
<CodeBlock code={`p.sendMessage("Hello World");`}/>
where the **p** is the first letter of the respective codeblock you want to use, followed by a dot, the actions name and its arguments. Actions must be followed by a semicolumn.
